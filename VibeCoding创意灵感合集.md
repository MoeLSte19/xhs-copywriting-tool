## 四、🎯 平台热点玩法方向（内容创作者参考）

### 小红书最热玩法
- **"30分钟搞定 XX"** —— 强调速度，反差感强
- **"不会编程但我用 AI 做了 XX"** —— 降门槛叙事，引发非技术人群共鸣
- **"我用 Vibe Coding 做了一个赚钱工具"** —— 变现刺激，转化率高
- **"晒我的 Vibe Coding 作品"** —— 对比展示，before/after 截图爆款模板
- **"黑客松 48 小时全程记录"** —— 过程感强，观众代入感高

### 抖音最热玩法
- **实操过程录屏** —— 屏幕录制从输入 Prompt 到出成品
- **"AI 秒变程序员"挑战** —— 从 0 到上线全流程
- **评论区晒作品** —— 抖音科技官方流量扶持，单条 5-10W 播放保底
- **作品 demo 演示视频** —— 直接展示产品交互效果
- **学生 / 小白成功案例** —— 情感共鸣强

### B站最热玩法
- **系统教程类**（如：零基础 7 天入门 Vibe Coding）
- **工具横评** —— Cursor vs Claude Code vs Trae
- **全程实录项目**（做 + 踩坑 + 上线全过程）
- **月入 XX 案例拆解** —— 数据驱动型内容
- **黑客松参赛记录 Vlog**

## 六、🚫 已证明无趣/没市场的方向（避坑清单）

| 方向 | 为什么不行 |
|------|---------|
| 做一个 CRUD Todo List | 同质化严重，毫无差异化，已是入门教程必备案例 |
| 复刻现有大平台 | 做不过大厂，没有护城河 |
| 纯技术炫技展示 | 非技术用户看不懂，技术用户觉得没意思 |
| 靠信息差倒卖账号 | 平台风险高，东方青案例已证明不可持续 |
| 过于复杂的大型系统 | Vibe Coding 不适合，开发质量不稳定 |
| 高安全要求的金融/医疗 App | AI 生成代码漏洞率高（约是人工的 1.7 倍） |

## 七、⚡ 直接可用的 Prompt 灵感模板

### 模板 1：工具型 App 启动 Prompt
```
用 [技术栈] 做一个 [功能描述] App。
要求：
- [核心功能 1]，[核心功能 2]，[核心功能 3]
- 界面风格：[参考风格，如"Apple Notes 极简风"]
- 支持移动端，响应式布局
- 数据本地存储，刷新不丢失
- 加上优雅的微动画
```

### 模板 2：美化/展示型页面
```
做一个 [类型] 个人主页，风格是 [描述风格]。
内容包括：[内容块 1]，[内容块 2]，[内容块 3]
要有 [动效要求]，颜色使用 [色调方向]。
要能截图直接分享，移动端优先。
```

### 模板 3：游戏型小应用
```
做一个 [游戏类型] 小游戏。
规则：[核心玩法描述]
界面：[风格描述，如"像素风/简洁白色/赛博朋克"]
支持 [分享/排行榜/再来一次] 功能
```

### 模板 4：让 AI 帮你挖掘需求（高级技巧）
```
我想构建 [简要描述]。
请采访我，问技术实现、UI/UX、边界情况和权衡。
不要问显而易见的问题，要深挖我没想到的难点。
把最终结论写成需求文档。
```
